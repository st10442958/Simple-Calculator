# 🧮 Simple Calculator (C++)

## Overview

This project is a menu-driven calculator developed in **C++**. It allows users to perform basic arithmetic operations through a simple console interface.

The application demonstrates fundamental C++ programming concepts including user input/output, conditional statements, switch-case selection, loops, and error handling.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero validation
* Menu-driven interface
* Option to perform multiple calculations
* Handles invalid menu selections

## Technologies Used

* C++
* Visual Studio 2022
* Standard C++ Library (`iostream`)

## Skills Demonstrated

* Variables and data types
* Input and output (`cin` and `cout`)
* Arithmetic operators
* `switch` statements
* Conditional statements (`if` / `else`)
* Loops (`do...while`)
* Basic error handling
* Console application development

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio.
3. Build the solution.
4. Run the application.
5. Select an operation from the menu.
6. Enter two numbers.
7. View the result.
8. Choose whether to perform another calculation or exit.

## Example

```text
=====================================
         SIMPLE CALCULATOR
=====================================
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit
=====================================
Enter your choice: 3

Enter first number: 8
Enter second number: 6

Result: 8 * 6 = 48
```
